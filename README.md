# gfaqs local

download gamefaqs guides for local access

## install

```
python3 -m venv venv
. venv/bin/activate
python3 -m pip install --requirement requirements.txt
```

## usage

```
/run.py download \
    --system-name gba \
    --game-name digimon \
    --game-name mega-man \
    --game-name metroid \
    --game-name pokemon \
    --game-name zelda
```
